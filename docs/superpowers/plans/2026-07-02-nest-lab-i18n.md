# NEST Lab i18n (KOR/ENG 토글) 구현 스펙

**목표:** 상단 KOR/ENG 토글로 한글↔영문 전환. 페이지는 한 벌 유지, 각 번역 지점에 한/영 텍스트를 함께 넣고 CSS로 표시 전환. 선택은 localStorage 기억.

## 메커니즘

1. **CSS** (`assets/css/style.css` 맨 끝에 추가):
```css
/* ===== 언어 전환 (KOR/ENG) ===== */
html[lang="en"] .lang-ko{display:none}
html[lang="ko"] .lang-en{display:none}
.lang-toggle{margin-left:4px;background:none;border:1px solid var(--line);color:var(--navy);
  font-weight:700;font-size:12px;padding:6px 10px;border-radius:7px;cursor:pointer;font-family:inherit;line-height:1}
.lang-toggle:hover{background:var(--tint)}
```
- 원리: `<html lang="ko">`가 기본(모든 페이지 이미 그러함) → `.lang-en`은 기본 숨김. 토글 시 `<html lang="en">` → `.lang-ko` 숨김.
- 번역 지점은 `<span class="lang-ko">한국어</span><span class="lang-en">English</span>` 형태로 감싼다. 숨김은 `display:none`만 쓰므로, 보이는 쪽은 원래 display(span=inline) 유지.

2. **JS** (`assets/js/main.js`):
- `DOMContentLoaded` 핸들러 **맨 처음**에 저장된 언어 적용:
  ```js
  document.documentElement.lang = localStorage.getItem('lang') || 'ko';
  ```
- `initNav()` 안에서 토글 버튼 배선(주입된 헤더에 있으므로 여기서):
  ```js
  const langBtn = document.querySelector('.lang-toggle');
  function syncLangBtn(){ langBtn && (langBtn.textContent = document.documentElement.lang === 'en' ? 'KOR' : 'ENG'); }
  syncLangBtn();
  langBtn && langBtn.addEventListener('click', ()=>{
    const next = document.documentElement.lang === 'en' ? 'ko' : 'en';
    document.documentElement.lang = next;
    localStorage.setItem('lang', next);
    syncLangBtn();
  });
  ```

3. **헤더** (`partials/header.html`):
- 브랜드 소제목을 이중 언어로:
  `<small><span class="lang-ko">나노바이오 융합 연구실</span><span class="lang-en">Nanobiomaterial Engineering Sensing Technology Lab</span></small>`
- 메뉴 `<ul class="menu">` 안, Contact `<li>` **다음**에 토글 버튼 추가:
  `<li><button class="lang-toggle" type="button" aria-label="언어 전환 / switch language">ENG</button></li>`
